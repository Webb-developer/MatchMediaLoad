MatchMediaLoad.js

Load images on a specified breakpoint to help reduce http requests until needed and page load speed.

View on GitHub Download .zip Download .tar.gz

Hi...

Basically, MatchMediaLoad is a plain javascript function that loads images only when they're ready to be seen. This saves bandwidth and HTTP requests and loading icons.

Browsers request images even if they are hidden, which you probably don't want. You don't want to load desktop only images on a mobile browser right??

So MatchMediaLoad takes an image with a media query as an attribute and loads that image only when the media query is met.
